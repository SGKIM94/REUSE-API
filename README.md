# REUSE
핸드폰을 중고 거래하기 위한 APP과 연동하기 위한 API 서버

## 개발 환경
 - Spring boot 
 - JPA(QueryDsl - mariaDB)
 - Junit5
 
## 서버 운용 환경
 - AWS EC2
 - RDS
 - S3
 
> ATDD 기반으로 개발되고 있습니다.  


# 현재 개발 진행중
 ```
 진행 상황
  - 사용자 기능 추가
  - 품목(이미지) 기능 추가
  - 카테고리 기능 추가
  - 채팅 기능 추가(WebSocket, STOMP)
  - 판매자에 대한 리뷰 기능 추가
  
 ```
